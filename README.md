## 📖 概要

2026年5月、Amazonはショッピング体験にAlexa Plusを統合し、Googleは25年間で最大の検索リデザインを発表した。どちらの発表にも、商品一覧ページを人間がスクロールして選ぶという行為は設計されていない。

デジタル広告市場は7,400億ドルを超える。Google（77%）、Meta（97%）、Amazon（第3の収益柱）——これらの企業の収益構造は「人間がクリックする」という行為に経済的に依存している。

本書の問いは明快だ。**AIエージェントが購買を代行する時、広告モデルはどうなるのか。**

過去の広告モデル移行では、注意の「表面」が変わった——紙からウェブへ、デスクトップからモバイルへ。しかし注意の「主体」は常に人間だった。エージェント商取引では、**注意の主体そのものが人間からAIに移行する**。人間がどこを見るかではなく、人間が見なくなる。

判定ではない。優劣の比較ではない。本書は、**現在進行している構造変化を、観測可能な事実とデータから記述する**ことを目的とするOSS書籍である。
